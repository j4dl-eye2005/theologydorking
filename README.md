# Theologia Operatorum
### A Google Dorking Cheat Sheet for Theological Research

*Theologia Operatorum* is a curated collection of Google Dorking commands for theological research. Instead of drowning in search results, use precision operators to surface hidden PDFs, open-access dissertations, seminary syllabi, biblical commentaries, and confessional documents from across Christianity, Judaism, Islam, and comparative religion. Covers patristics, Reformation texts, Koine Greek lexicons, mystical theology, and more — with one-click Google links for every command.

---

## What's Inside

- **Sacred Document Retrieval** — PDFs of patristic writings, Reformation treatises, biblical commentaries, Dead Sea Scrolls, and apocrypha
- **Academic Repositories** — targeted searches on JSTOR, Academia.edu, seminary `.edu` domains, and open dissertations
- **Ancient Languages** — Koine Greek lexicons, Biblical Hebrew grammars, Septuagint studies
- **By Tradition** — Catholic (Vatican.va), Eastern Orthodox, Reformed confessions, Liberation theology, Jewish mysticism, Islamic kalam
- **Advanced Combinations** — seminary syllabi, Internet Archive books, PhilPapers, open Bible datasets

---

## How to Use

Open `theology-dorking.html` in any browser. Each command card includes:

- A ready-to-copy search string with color-coded operators
- A description of what it finds
- A direct **↗ Open in Google** link to run it instantly

Click the **COPY** button on any card to copy the raw command to your clipboard, then paste it into Google.

---

## Operators Used

| Operator | Purpose |
|----------|---------|
| `filetype:` | Restrict results to a specific file type (e.g. `pdf`) |
| `site:` | Search within a single domain |
| `intitle:` | Match a word in the page title |
| `inurl:` | Match a word in the URL |
| `intext:` | Match a word in the body text |
| `"quotes"` | Exact phrase match |
| `-word` | Exclude a term from results |
| `OR` | Match either term (must be capitalized) |

---

## Tips

- Combine operators freely: `filetype:pdf site:.edu "pneumatology"` sharpens results considerably.
- Add `after:2010 before:2023` to filter by publication year.
- Use parentheses to group alternatives: `("term1" OR "term2")`.
- For rare manuscripts, try adding `intext:"rare"` or `intext:"manuscript"` to your query.
